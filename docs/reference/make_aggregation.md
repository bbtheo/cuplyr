# Create an aggregation structure for summarise

Create an aggregation structure for summarise

## Usage

``` r
make_aggregation(output_col, input_col, fn, input_type = NULL)
```

## Arguments

- output_col:

  Output column name

- input_col:

  Input column name

- fn:

  Aggregation function name (sum, mean, min, max, n)

- input_type:

  Input column type

## Value

An aggregation list structure
